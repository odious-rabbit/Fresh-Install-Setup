# uBlock Origin Configuration (Firefox)

*IF YOU USE PHOENIX WITH FIREFOX THIS SETUP SHOULD ALREADY BE CONFIGURED*

## Setup

1. Install the [uBlock Origin extension](https://addons.mozilla.org/en-US/firefox/addon/ublock-origin/).
2. Import the settings file (`ublock-origin-settings.txt`) via uBlock Origin's Dashboard → Settings → "Restore from file".
